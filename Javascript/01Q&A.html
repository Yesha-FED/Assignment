Q.1 What is Javascript?

. JavaScript is a high-level interpreted programming language.

. It is one of the three core technologies of the World Wide Web, along with HTML and CSS.

. JavaScript is used to make web pages interactive and to add dynamic content.

. It can also be used to create games, animations, and other multimedia applications.

Q.7 What is the difference between ViewState and SessionState?

. ViewState and SessionState are both mechanisms for maintaining state in ASP.NET applications.

. However, there are some key differences between the two.

. ViewState is maintained on the client side, while SessionState is maintained on the server side.

. This means that ViewState is stored in the browser, while SessionState is stored in memory on the server.

Q.11 What are all the looping structures in JavaScript?

Q.14 What are all the types of Pop up boxes available in JavaScript?

1. Alert box: This is the simplest type of pop-up box, and it is used to display a message to the user. The user must click the "OK" button to close the box.

2. Confirm box: This type of pop-up box is used to ask the user to confirm an action. The user can click the "OK" button to confirm the action, or the "Cancel" button to cancel it.

3. Prompt box: This type of pop-up box is used to prompt the user for input. The user can type text into the box and then click the "OK" button to submit the input.
