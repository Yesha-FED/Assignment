Q40.0 Write a JavaScript Program to display the current day and time in the following format. Sample Output: Today is Friday. Current Time is 12 PM: 12 : 22 2 ?

script
let date = new Date();
let days = ['Sunday', 'Monday', 'Tuseday', 'Wendnesday', 'Thursday', 'Friday', 'Saturday'];
let dat = date.getDay();
let time = date.getHours();
console.log(`Today is ${days[dat]}.Current time is ${time > 12 ? time - 12 : time} ${time >= 12 ? 'PM' : 'AM'}`);
script
