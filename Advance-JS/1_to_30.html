Q8. Write a JavaScript Program to find the area of a triangle?

script
/*Define the lengths of the three t of a triangle */
let y1 = 2;
let y2 = 3;
let y3 = 4;
/* Calculate the semi-perimeter of the triangle */
let s = (y1 + y2 + y3) / 2;
/* Use Heron's formula to calculate the area of the triangle */
let area = Math.sqrt(s * ((s - y1) * (s - y2) * (s - y3)));
/* Log the calculated area to the console */
console.log(area);
script
